# 🧮 Smart Calculator in C

A menu-driven Smart Calculator built using the C programming language. This project performs basic and advanced mathematical operations while maintaining a history of all calculations using file handling.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔢 Factorial
- 💪 Power Calculation
- ✅ Prime Number Check
- 📌 GCD (Greatest Common Divisor)
- 📜 View Calculation History
- 🗑️ Clear History

---

## 🛠️ Concepts Used

- Functions
- Switch Case
- Loops
- Conditional Statements
- Strings
- File Handling
- Math Library (`math.h`)

## ▶️ How to Compile and Run

### Compile

```bash
gcc main.c -o calculator -lm
```

### Run

**Windows**

```bash
calculator.exe
```

**Linux / macOS**

```bash
./calculator
```
